Pi-Jukebox
==========

A Raspberry Pi jukebox powered by Spotify

## Jukebox API

### Spotify Track Search

**GET:** `/api/search/spotify/track`

**Parameters:**

* `q`: Search string (required)
* `page`: The page of the result set to return (optional - defaults to 1)
